API REST pour la gestion des mémoires de fin d'études

Endpoints API Disponibles

GET
/api/tracks/
Récupérer toutes les filières
Retourne la liste complète des tracks/filières disponibles
POST
/api/tracks/create/
Créer une nouvelle filière
Ajoute une nouvelle track/filière à la base de données
GET
/api/tracks/<int:track_id>/
Récupérer une filière par ID
Retourne les détails d'une track/filière spécifique
PATCH
/api/tracks/update/<int:track_id>/
Mettre à jour une filière
Met à jour partiellement les informations d'une track/filière
DELETE
/api/tracks/delete/<int:track_id>/
Supprimer une filière
Supprime une track/filière de la base de données
GET
/api/memoires/
Récupérer tous les mémoires
Retourne la liste complète des mémoires
GET
/api/encadreurs/
Récupérer tous les encadreurs
Retourne la liste complète des encadreurs
